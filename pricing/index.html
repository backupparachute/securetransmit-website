Every plan sets a default file retention window, 7 days on Solo, 30 on Pro, 90 on Business, fully custom on Enterprise. When that window expires, the file data is deleted. Audit records and SHA-256 hashes are retained separately (1 year on Solo, forever on Pro and up) so you can still prove what moved, even after the file is gone.

The moment the destination system acknowledges receipt, the file is deleted. No retention window, no waiting. The SHA-256 hash and the audit record stay forever. Breach exposes only in-flight data. Available on Pro and up. Default on Enterprise if you want it.

Island mode is architectural, not metered. It's included on Pro and up at no extra charge. Partners push to SecureTransmit, your systems pull from SecureTransmit, and you close every inbound B2B port on your own firewall.
